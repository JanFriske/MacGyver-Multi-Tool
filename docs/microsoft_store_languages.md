# Microsoft Store – Unterstützte Sprachen

## Übersicht

Der Microsoft Store unterstützt **über 110 Sprachen** für App-Einreichungen. Apps müssen mindestens eine dieser Sprachen unterstützen.

**Quelle:** Microsoft Learn Documentation  
**Standard:** BCP-47 Language Tags  
**Stand:** 23. November 2024

## Hinweise

1. **Mindestanforderung:** Apps müssen mindestens eine dieser Sprachen unterstützen
2. **BCP-47 Standard:** Alle Sprachcodes folgen dem BCP-47-Standard
3. **Regionale Varianten:** Einige Sprachen haben regionale Varianten (z.B. en-US, en-GB)
4. **Schriftsysteme:** Manche Sprachen werden in verschiedenen Schriften angeboten (z.B. sr-Cyrl, sr-Latn)
5. **Store-Listing:** Die im App-Manifest angegebenen Sprachen werden Kunden im Microsoft Store angezeigt
